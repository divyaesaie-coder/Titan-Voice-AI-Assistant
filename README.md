# Titan Voice AI Assistant

Titan Voice AI Assistant is a desktop-based AI assistant developed using Python. It combines voice recognition, text-to-speech, and Google Gemini AI to provide intelligent voice interactions through a simple graphical interface.

## Features

- Voice Input using Speech Recognition
- Voice Output using Windows Speech API
- Google Gemini AI Integration
- Interactive Tkinter GUI
- Time and Date Commands
- General Knowledge Question Answering
- AI-Powered Conversations
- Single-Click Voice Interaction

## Technologies Used

- Python
- Tkinter
- SpeechRecognition
- PyAudio
- Google Gemini AI
- pywin32

## How It Works

1. User clicks the Speak button.
2. Assistant listens to the user's voice.
3. Speech is converted into text.
4. The query is processed.
5. Built-in commands are handled locally.
6. Other queries are sent to Gemini AI.
7. The response is displayed and spoken aloud.

## Installation

Install required packages:

pip install SpeechRecognition pyaudio pywin32 google-generativeai

## Run

python main.py
